# Устранение проблем

## Проблема: Страница не загружается или показывает черный экран

### Решение 1: Перезапуск dev сервера

1. Остановите текущий процесс (Ctrl+C в терминале)
2. Удалите папку `.next`:
   ```bash
   Remove-Item -Recurse -Force .next
   ```
3. Перезапустите сервер:
   ```bash
   npm run dev
   ```

### Решение 2: Проверка консоли браузера

Откройте DevTools (F12) и проверьте вкладку Console на наличие ошибок.

### Решение 3: Проверка импортов

Убедитесь, что все импорты используют правильные пути:
- `@/components/...` - для компонентов
- `@/lib/...` - для утилит

### Решение 4: Проверка CSS переменных

Все цвета теперь используют произвольные значения Tailwind:
- `bg-[var(--color-primary)]` вместо `bg-primary`
- `text-[var(--color-stone)]` вместо `text-stone`

Это необходимо для работы с Tailwind CSS 4.

## Проблема: Видео не загружается

Видео для Hero секции должно быть в `/public/videos/hero-moss.mp4`. Если файла нет, будет показан градиентный фон.

## Проблема: Шрифты не применяются

Шрифты настроены через Google Fonts в `layout.tsx`. Убедитесь, что:
1. Интернет-соединение активно (для загрузки шрифтов)
2. CSS переменные `--font-heading` и `--font-body` определены

## Частые ошибки

### Module not found
- Проверьте правильность путей импорта
- Убедитесь, что файлы существуют

### CSS классы не работают
- Используйте произвольные значения: `bg-[var(--color-primary)]`
- Или добавьте классы в `tailwind.config.ts`
