# Логотипы TLEYOU

В этой папке находятся файлы логотипа:

- **tleyou-logo-white.svg** - белый логотип (для темных фонов)
- **tleyou-logo-black.svg** - черный логотип (для светлых фонов)

## Текущие файлы:
Созданы базовые SVG версии логотипа. Вы можете заменить их на PNG версии с вашим финальным дизайном.

## Если хотите использовать PNG вместо SVG:
1. Сохраните файлы как `tleyou-logo-white.png` и `tleyou-logo-black.png`
2. Обновите компонент `Logo.tsx`, заменив `.svg` на `.png` в путях

## Использование:
Компонент `Logo` автоматически выбирает правильный вариант в зависимости от фона:
- `variant="light"` - белый логотип (для темных фонов)
- `variant="dark"` - черный логотип (для светлых фонов)
