# Видео для Hero секции

Поместите видео файл `hero-moss.mp4` в эту папку.

## Требования к видео:

- **Формат**: MP4
- **Разрешение**: Минимум 1920x1080 (Full HD)
- **Длительность**: 10-30 секунд (будет зациклено)
- **Содержание**: Мох + туман, атмосферная съёмка
- **Размер файла**: Рекомендуется до 10MB (используйте сжатие)

## Оптимизация:

Для уменьшения размера файла используйте:
```bash
ffmpeg -i input.mp4 -vcodec libx264 -crf 28 -preset slow -an hero-moss.mp4
```

Где:
- `-crf 28` - качество (меньше = лучше качество, больше = меньше размер)
- `-an` - удаляет аудио (не нужно для фона)
